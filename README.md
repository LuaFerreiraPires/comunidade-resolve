# Comunidade Resolve

## Descrição
O Comunidade Resolve é uma aplicação web desenvolvida com o objetivo de permitir que usuários registrem, acompanhem e gerenciem problemas encontrados em sua comunidade, como questões de infraestrutura, segurança e limpeza.

A aplicação foi desenvolvida como parte de um trabalho acadêmico, aplicando conceitos de desenvolvimento web, metodologias ágeis e integração com serviços em nuvem.

## Funcionalidades

- Cadastro de usuário (e-mail e senha)
- Login e autenticação com Firebase
- Registro de problemas
- Edição de problemas
- Exclusão de problemas
- Marcar problema como resolvido
- Upload de imagem
- Filtro por status (Pendente / Resolvido)
- Filtro por categoria (Infraestrutura, Segurança, Limpeza)
- Logout do sistema

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Firebase Authentication
- Firebase Firestore
